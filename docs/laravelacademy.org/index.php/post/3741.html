公司简介

传太医在2016年初拿到了数千万A轮投资，有着强大的医疗圈背景和百度系的投资人资源。线上技术团队有来自IBM、阿里、360、豆瓣、美团、新浪等技术牛人。此外，我们还有以下标签：

  • 遵循工程师文化，效率文化，在这里你可以和身边的工程师牛人讨论技术，相互学习
  • Mac化办公，希望你自己也有一台高配的Mac，HHKB，Realforce键盘什么的
  • 各种IT基础设施加持，神马x墙啊，一键Docker启动开发环境啊，我们都有
  • React前端大牛帮你搞定客户端需求，终于可以安心做后端了。什么，你想做全栈？师傅在这里，剩下看你自己的了
  • 重度代码洁癖，烂代码不能忍
  • 奋斗的氛围，能者多劳，劳者多得
  • 5月就会搬到高大上的700平写字楼里，目前正在装修中

公司位置

公司在北二环地铁站附近，交通便利

岗位职责

  • 根据业务需求实现功能，抽象和封装业务代码，保证代码的可扩展性
  • 封装PHP应用组件，开发Laravel扩展
  • 完成大型项目和系统的开发，承担技术调研等前瞻工作

岗位要求

  • 3年以上LNMP开发经验，有大型系统开发经验
  • SQL熟练，Linux操作熟练
  • 掌握SQL优化，PHP优化，缓存优化等知识
  • 运维部署相关知识，能够在分布式环境中定位问题，修复bug
  • 有Laravel开发经验
  • 对新事物保持开放学习的态度

薪酬范围

20k~35k

联系方式

欢迎想通过自己的努力实现弯道超车的同学，也许你错过了小米、滴滴、饿了么，这一次不要错过，机会总是留给那些有准备的人，公司福利随着每次融资都会有较大的提升。起码来公司聊聊技术嘛~

有意者欢迎联系邮箱 yangliu@chuanty.com
